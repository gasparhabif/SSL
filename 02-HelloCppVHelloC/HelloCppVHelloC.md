# Hello World en C++
```
#include <iostream>

using namespace std;

int main() {
    cout << "Hello World!" << endl;
    return 0;
}
```

# Hello World en C

```
#include <stdio.h>

int main() {
    printf("Hello World!\n");
}
```

# Hello World en Java

```
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World");
    }
}
```
